## Round

```math
Round(n) = f(n) = Floor((n mod 1) + 0.5)

Round(1.2) = 1
Round(2.8) = 3
```

## Round up

```
Ceil(n) = f(n) = n + 1 - (n mod 1)

Ceil(1.2) = 2
Ceil(2.8) = 3
```

## Round down

```
Floor(n) = f(n) = n - (n mod 1)

Floor(1.2) = 1
Floor(2.8) = 2
```
